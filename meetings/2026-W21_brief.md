## 2. 핵심 결과 / 발견

- **문서화 완성도:** PatchCore 재현 실험의 모든 과정(재현, 실패 분석, 검증 실험, 환경 계획)이 문서화되어 자기-완결적 학습 단위로 구성됨.
- **재현성 인사이트:** `pill`과 `metal_nut`의 성능 편차가 단순 환경 차이를 넘어 데이터 특성(샘플링 효율, 해상도 의존성)에 기인함을 명확히 함.
- **SimpleNet 1차 재현 성공:** toothbrush 카테고리에서 I-AUROC 1.000 / P-AUROC 0.983 — 단일 카테고리지만 SimpleNet 논문 평균(99.6%)과 부합. PatchCore가 saturating한 카테고리에서 SimpleNet도 saturating 확인. 다만 anomaly pixel AUROC 0.904는 localization 정밀도 관점에서 다른 카테고리 거동 확인 필요.

## 3. 막힌 부분 / 의문점

- **환경 재현성 우선순위:** Docker 도입을 통한 완벽한 환경 고정을 PatchCore 마무리 단계에서 수행할지, 아니면 차세대 알고리즘(SimpleNet 등) 시작 시점의 베이스라인 구축으로 넘길지 결정 필요.
- **SimpleNet 확장 카테고리 선택:** method1과 직접 비교를 위해 어떤 카테고리부터 확장할지 — PatchCore가 강했던 카테고리(bottle, leather) vs. 격차가 있었던 카테고리(pill, metal_nut). 후자에서 SimpleNet이 우위라면 흥미로운 발견.
